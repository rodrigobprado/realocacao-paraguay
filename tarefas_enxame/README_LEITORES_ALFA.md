# Tarefas Leitores Alfa

Backlog adicional para cobrir as lacunas apontadas pelos leitores alfa.

## Escopo
- 6 tarefas de nivel `pais`
- 18 tarefas de nivel `departamento`
- 262 tarefas de nivel `localidade`
- Total: 286 tarefas

## Como usar
- Execute primeiro as tarefas `ALF-PAIS-*`.
- Depois execute as tarefas `ALF-DEP-*`, que dependem das tarefas de pais.
- Em seguida execute as tarefas `ALF-LOC-*`, cada uma dependente do respectivo departamento.
- Cada tarefa deve gerar conteudo pesquisavel e citavel para futura incorporacao no livro.

## Observacoes de escopo
- Este backlog nao substitui `TAREFAS_ENXAME.csv`; ele complementa o livro com lacunas percebidas em leituras alfa.
- A prioridade padrao e `P1` para pais e departamento, e `P2` para localidade, por volume de execucao.
- Quando a fonte oficial nao existir, registrar a lacuna de forma explicita e procurar proxy oficial de nivel superior.
